# 环境搭建

运行环境是在 wsl2 的 Ubuntu 20.04 LTS 的基础上搭建的 Java 环境。

首先更新库

```sh
sudo apt update
```

然后安装 Java 的 JDK 16

```sh
sudo apt install openjdk-16-jdk
```

安装完后检查版本

```sh
java -version
```
